# MRI Fingerprinting: Recovery with Complex-Valued Neural Networks

Данный проект посвящен разработке и обучению комплексно-значной нейронной сети для восстановления количественных карт магнитно-резонансной томографии (T1, T2 и B0) на основе технологии **Magnetic Resonance Fingerprinting (MRF)**.

## 📌 Описание проекта

Метод MR-фингерпринтинга позволяет отойти от качественного анализа изображений к количественному измерению физических параметров тканей. В рамках проекта реализован полный цикл: от симуляции физики сигнала до разработки нейросетевой архитектуры, способной работать в комплексном домене.

**Основные компоненты:**
* **Bloch Simulator:** Численное решение уравнений Блоха для генерации обучающих данных.
* **Complex-Valued NN:** Нейронная сеть, использующая комплексно-значные веса и активацию **Complex Cardioid**.
* **Wirtinger Optimizer:** Реализация градиентного спуска с использованием исчисления Виртингера.

## 🧬 Математическая модель

### Уравнения Блоха и сигнал МРТ
Генерация сигналов (фингерпринтов) базируется на уравнениях Блоха, описывающих эволюцию вектора макроскопической намагниченности $\vec{M}$ под действием магнитного поля $\vec{B}$:
$$\frac{d\vec{M}}{dt} = \gamma \vec{M} \times \vec{B}$$
Для моделирования релаксации используются параметры $T_1$ (продольная) и $T_2$ (поперечная). Для оптимизации сигнала в условиях короткого времени повторения ($TR$) используется расчет **угла Эрнста**:
$$\alpha_{Ernst} = \arccos(e^{-TR/T_1})$$

### Обучение в комплексном домене
Поскольку МРТ-данные представлены комплексными числами ($A + iB$), стандартные методы оптимизации неприменимы из-за нарушения условий Коши-Римана. Для обучения используется **исчисление Виртингера**, где веса обновляются через коградиент по сопряженным переменным:
$$w_{n+1} = w_n - \alpha \nabla_{\bar{w}} L$$
Это позволяет сети одновременно оптимизировать и амплитуду, и фазу сигнала.
